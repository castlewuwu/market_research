# 中国大陆新能源汽车市场调研报告

> 调研日期：2026年3月21日
> 数据来源：行业报告、公开数据

---

## 一、市场概况

### 1.1 市场规模

| 年份 | 销量（万辆） | 同比增长 | 渗透率 |
|------|-------------|---------|--------|
| 2020 | 136.7 | +10.9% | 5.4% |
| 2021 | 352.1 | +157.6% | 13.4% |
| 2022 | 688.7 | +95.6% | 25.6% |
| 2023 | 949.5 | +37.9% | 31.6% |
| 2024 | 1285.0 | +35.3% | 40.2% |
| 2025 | 1649.0 | +28.2% | ~50% |
| 2026E | 1900.0 | +15.2% | ~55% |

**关键数据：**
- 2025年中国新能源汽车销量达**1649万辆**
- 市场渗透率接近**50%**，突破临界点
- 出口量**261.5万辆**，同比翻倍
- 国内乘用车增速放缓至16.17%

### 1.2 全球地位

| 维度 | 中国 | 全球占比 |
|------|------|---------|
| 产销规模 | 1649万辆 | ~60% |
| 出口量 | 261.5万辆 | 全球第一 |
| 充电桩数量 | 800万+ | ~70% |
| 动力电池产能 | 700GWh+ | ~70% |

## 二、竞争格局

### 2.1 2025年品牌销量排名

| 排名 | 品牌 | 年销量（万辆） | 市场份额 | 同比变化 |
|------|------|---------------|---------|---------|
| 1 | 比亚迪 | ~420 | 25.5% | +15% |
| 2 | 特斯拉中国 | ~95 | 5.8% | +8% |
| 3 | 理想汽车 | ~50 | 3.0% | +25% |
| 4 | 蔚来汽车 | ~25 | 1.5% | +10% |
| 5 | 小鹏汽车 | ~20 | 1.2% | +5% |
| 6 | 零跑汽车 | ~30 | 1.8% | +50% |
| 7 | 小米汽车 | ~15 | 0.9% | 新进 |
| 8 | 华为问界 | ~40 | 2.4% | +60% |
| 9 | 吉利新能源 | ~80 | 4.9% | +35% |
| 10 | 奇瑞新能源 | ~60 | 3.6% | +40% |

**关键发现：**
- 比亚迪稳居第一，但市场份额承压
- 新势力分化明显：理想、零跑表现亮眼
- 小米汽车首年销量约15万辆，表现超预期
- 华为生态深度渗透，问界系列增长强劲

### 2.2 细分市场格局

**按技术路线：**

| 类型 | 市场份额 | 趋势 |
|------|---------|------|
| 纯电（BEV） | 58% | 增速反超插混 |
| 插混（PHEV） | 28% | 稳定增长 |
| 增程（EREV） | 14% | 理想带动增长 |

**按价格段：**

| 价格段 | 市场份额 | 主要品牌 |
|--------|---------|---------|
| 10万以下 | 20% | 五菱、比亚迪（海鸥）、零跑 |
| 10-20万 | 35% | 比亚迪、吉利、埃安 |
| 20-30万 | 25% | 特斯拉、理想、小鹏 |
| 30万以上 | 20% | 理想、蔚来、问界、小米 |

## 三、重点品牌分析

### 3.1 比亚迪

| 维度 | 信息 |
|------|------|
| 市场地位 | 中国新能源汽车销量第一 |
| 年销量 | 约420万辆（2025年） |
| 核心优势 | 全产业链布局、刀片电池、DM-i技术 |
| 产品矩阵 | 王朝系列、海洋系列、腾势、仰望 |
| 出海战略 | 已进入100+国家，海外销量快速增长 |
| 挑战 | 国内竞争加剧，市场份额承压 |

### 3.2 特斯拉中国

| 维度 | 信息 |
|------|------|
| 市场地位 | 外资品牌第一 |
| 年销量 | 约95万辆（中国区） |
| 核心优势 | 品牌影响力、FSD自动驾驶、超级充电网络 |
| 产品系列 | Model 3、Model Y、Model S、Model X |
| 本土化 | 上海超级工厂，本土化率超95% |
| 挑战 | 国产品牌冲击，价格战压力 |

### 3.3 理想汽车

| 维度 | 信息 |
|------|------|
| 市场地位 | 新势力第一 |
| 年销量 | 约50万辆 |
| 核心优势 | 增程技术、家庭用户定位、智能化 |
| 产品系列 | L系列（L6/L7/L8/L9）、MEGA |
| 财务表现 | 2025年净利润11亿元，连续三年盈利 |
| 挑战 | 纯电转型、MEGA销量不及预期 |

### 3.4 华为生态（问界/智界）

| 维度 | 信息 |
|------|------|
| 市场地位 | 智能化第一梯队 |
| 年销量 | 约40万辆 |
| 核心优势 | 华为品牌、鸿蒙智行、ADS智驾 |
| 产品系列 | 问界M5/M7/M9、智界S7 |
| 合作模式 | HI模式+智选车模式 |
| 挑战 | 产能爬坡、渠道建设 |

### 3.5 小米汽车

| 维度 | 信息 |
|------|------|
| 市场地位 | 新晋黑马 |
| 年销量 | 约15万辆（首年） |
| 核心优势 | 小米生态、粉丝经济、性价比 |
| 产品系列 | SU7系列 |
| 亮点 | 首年销量超预期，品牌影响力强 |
| 挑战 | 产能受限、产品线单一 |

## 八、政策环境

### 8.1 政策支持

| 政策 | 内容 |
|------|------|
| 购置税减免 | 新能源汽车购置税减免至2027年 |
| "双新"补贴 | 以旧换新补贴 |
| 充电设施建设 | 加快充电桩建设 |
| 碳积分政策 | 双积分制度持续 |

### 8.2 政策趋势

- 2026年补贴逐步退坡
- 重点转向基础设施建设和充电便利性
- 碳中和目标推动长期发展
- 智能化标准制定加速

## 十、投资与发展建议

### 10.1 品牌建议

**对于比亚迪：**
1. 持续技术创新，保持领先
2. 加速出海布局
3. 高端品牌仰望突破

**对于新势力：**
1. 理想：加速纯电转型
2. 蔚来：提升盈利能力
3. 小鹏：差异化竞争
4. 零跑：性价比路线巩固

**对于传统车企：**
1. 加速新能源转型
2. 利用现有渠道优势
3. 细分市场突破

### 10.2 市场机会

| 机会 | 说明 |
|------|------|
| 固态电池 | 下一代技术革命 |
| 智能化 | 城市NOA、高阶智驾 |
| 出海 | 海外市场空间大 |
| 下沉市场 | 三四线城市渗透率提升 |
| 商用车 | 电动化、智能化改造 |
